# Project 1: Question Page

The project makes a question interface, implementing the question data from the BoilerExam API. It uses KaTeX to handle the latex and gives user feedback when submitted. 

## Tech Stack

- Framework: React
- Styling: Tailwind CSS
- Math: KaTeX

## Install

Prequisites: Have node.js installed

#### Installation:

1. Clone the repository:

`git clone https://github.com/knguyen271/boilerexam_take_home.git`

2. Change to the project directory

`cd boilerexam_take_home`

3. Install dependencies:

`npm install`


#### Running the Project:

`npm run dev`

Open your browser and navigate to the local URL provided in the terminal (http://localhost:3000).
